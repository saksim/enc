# v0.1.0-mainline-beta.1 Release Notes

发布日期：2026-06-13

## Release Verdict

非 OCR 加密 / 代码保护能力已进入 Mainline Beta 上线状态。

本版本只声明非 OCR 主线能力，不声明 OCR、真实拍照扫描、跨介质传输或 remote-KMS 已正式上线。

## Shipped Capability

当前可发布主线：

```text
protect -> build -> package -> verify -> release
```

用户可以依赖的能力：

```text
任意文件加密保护：file -> encrypted artifact -> decrypt/verify
Python 代码保护：.py -> encrypted staging -> .pyd/.so -> import/run
发布治理闭环：signed manifest -> signed release approval -> promotion evidence -> artifact bundle -> non-OCR release gate
```

## Evidence Boundary

已纳入发布门禁的证据链：

```text
license-file production default
bundle_license = false
manifest signature required
native-only runtime loader
runtime integrity checks
release approval required
promotion artifact bundle required
non-OCR release gate required
protected ref / production-promotion environment gate
```

历史验证记录：

```text
非 OCR 主线测试：127 passed, 7 skipped
Code Protection strict native smoke：CODE_PROTECTION_SMOKE_OK
Runtime integrity smoke：RUNTIME_INTEGRITY_SMOKE_PASSED
```

## Product Promise

可以对外承诺：

```text
避免源码明文直接暴露
显著提高逆向、破译、篡改成本
保护任意文件数据
Python 代码加密保护后可 native packaging 并保持运行能力
```

不能对外承诺：

```text
绝对不可逆向
绝对不可破解
SO/PYD 等同强保密边界
local-embedded 可抵抗强攻击者
攻击者拿到完整运行环境和密钥后仍无法恢复逻辑
OCR / cross-media transfer 已随本版本上线
```
