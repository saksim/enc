# v0.1.0-ga Release Notes

发布日期：2026-06-13
状态：GA

## Release Verdict

非 OCR 加密 / 代码保护能力已从 Mainline Beta 推进到 GA 发布状态。

本版本只声明非 OCR 主线能力，不声明 OCR、真实拍照扫描、跨介质传输、remote-KMS 服务端或第三方逆向评估报告已正式上线。

## What Changed

本次 GA 相比 `v0.1.0-mainline-beta.1` 补齐了正式发布治理闭环：

```text
发布证据归档
approval key 轮换演练
生产 license-file E2E fail-closed 示例
Release 包 reverse-cost / no-source-leak 硬门禁
GA 用户手册与版本事实文档
```

## What Users Can Rely On

当前 GA 可依赖能力：

```text
任意文件加密、解密、完整性校验
Python 代码加密保护后 native packaging 为 .pyd/.so，并保持 import/run 能力
生产默认 license-file 外置交付
license 签名、机器绑定、过期、吊销 fail-closed
manifest signature
runtime integrity fail-closed
release approval / receipt / tamper report
promotion artifact bundle
non-OCR release gate
release dist no-source-leak check
```

## Product Promise

可以对外声明：

```text
避免源代码明文直接暴露
显著提高低成本逆向、破解和篡改成本
支持任意文件加密保护
支持 Python 代码保护后 native packaging 并保持运行能力
支持 release 证据归档、签名审批、promotion artifact bundle 和 tamper report
```

不能对外声明：

```text
绝对不可逆向
绝对不可破解
SO/PYD 等同强保密边界
local-embedded 可抵抗强攻击者
攻击者拿到完整运行环境和密钥后仍无法恢复逻辑
OCR / cross-media transfer 已随本版本上线
remote-KMS 服务端已随本版本正式上线
```
